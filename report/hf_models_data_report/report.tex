\section{Data Source}

\begin{itemize}
  \item \textbf{Source:} Hugging Face Hub model metadata (public API)
  \item \textbf{Format:} JSON Lines, fetched via \texttt{huggingface\_hub} Python library
  \item \textbf{Size:} 2,815,064 models, 1.7\,GB JSONL (229\,MB gzipped)
  \item \textbf{Supplementary:} 2,019,739 evaluation results (580\,MB JSONL)
  \item \textbf{Spark code:} Scala 2.12, packaged as JAR via sbt, executed on NYU Dataproc
\end{itemize}

\section{Ingestion}

Two stages:

\begin{enumerate}
  \item \textbf{API Fetch} (Python): \texttt{ingest\_hf\_models.py fetch} downloads all public models via \texttt{HfApi.list\_models()} with \texttt{expand} for \texttt{safetensors}, \texttt{config}, and \texttt{cardData}. Outputs \texttt{hf\_models.jsonl} and \texttt{hf\_eval\_results.jsonl}.
  \item \textbf{Spark Ingestion} (Scala): \texttt{IngestHfModels} reads JSONL with explicit schema, writes raw Parquet.
\end{enumerate}

\begin{lstlisting}[language=bash,caption={Commands run on NYU Dataproc}]
# Upload to HDFS
hdfs dfs -cp gs://nyu-dataproc-hdfs-ingest/oss-pulse/hf_models.jsonl.gz \
  /user/jl17797_nyu_edu/oss_pulse/source/huggingface_hub/

# Ingestion
spark-submit --class osspulse.ingestion.IngestHfModels \
  oss-pulse_2.12-0.1.0.jar \
  --input /user/jl17797_nyu_edu/oss_pulse/source/huggingface_hub/hf_models.jsonl.gz \
  --raw-output /user/jl17797_nyu_edu/oss_pulse/raw/huggingface_hub
\end{lstlisting}

\section{Data Cleaning}

Scala class: \texttt{osspulse.cleaning.CleanHfModels}. Steps:

\begin{enumerate}
  \item Filter out private and disabled models.
  \item Split \texttt{modelId} into \texttt{owner} and \texttt{model\_name}.
  \item Normalize \texttt{pipeline\_tag} and \texttt{library\_name} (lowercase, null $\to$ \texttt{"unknown"}).
  \item Parse timestamps (\texttt{lastModified} $\to$ \texttt{last\_modified\_ts}).
  \item Extract and normalize \texttt{license} from \texttt{card\_data} with synonym mapping.
  \item Extract \texttt{parameter\_count} from \texttt{safetensors} and \texttt{model\_type} from \texttt{config}.
  \item Filter records with null \texttt{modelId} or invalid format.
  \item Deduplicate by \texttt{model\_id}.
\end{enumerate}

Total cleaned records: \textbf{2,815,064}.

\begin{lstlisting}[language=bash,caption={Profiling and cleaning on NYU Dataproc}]
spark-submit --class osspulse.profiling.ProfileHfModels \
  oss-pulse_2.12-0.1.0.jar \
  --input /user/jl17797_nyu_edu/oss_pulse/raw/huggingface_hub

spark-submit --class osspulse.cleaning.CleanHfModels \
  oss-pulse_2.12-0.1.0.jar \
  --input /user/jl17797_nyu_edu/oss_pulse/raw/huggingface_hub \
  --output /user/jl17797_nyu_edu/oss_pulse/cleaned/huggingface_hub
\end{lstlisting}
